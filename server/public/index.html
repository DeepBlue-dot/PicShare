PinClone API

A Pinterest-like REST API built with Node.js, Express & MongoDB

Node.js Express MongoDB JWT Auth

📚 Project Overview

PinClone is a feature-rich RESTful API that powers Pinterest-like applications. Key features include:

  • 🔐 JWT Authentication & Authorization
  • 🖼️ Image Uploads with Cloudinary Integration
  • 💬 Comments & Likes System
  • 📌 Board Organization & Collaboration
  • 🔍 Search & Filtering
  • 📈 Pagination & Rate Limiting

🚀 Quick Start

curl -X POST https://api.pinclone.com/posts \
-H "Authorization: Bearer YOUR_TOKEN" \
-F "title=Sunset" \
-F "description=Beautiful beach sunset" \
-F "image=@sunset.jpg"
const response = await fetch('https://api.pinclone.com/posts', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: formData
});
import requests

response = requests.post(
    'https://api.pinclone.com/posts',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    files={'image': open('sunset.jpg', 'rb')},
    data={'title': 'Sunset', 'description': 'Beautiful beach sunset'}
)
